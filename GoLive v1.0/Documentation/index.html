GoLive - Responsive HTML5 Template

  • Created on: 23/2/2017
  • by: nilanjanbanerjee

Thank you for purchasing GoLive.
“GoLive” is a Corporate Template which have clean & modern with Responsive design.

HTML

This template, which is a responsive layout for tablet and other mobile devices, is based on the HTML5 structure and Twitter Bootstrap 3. Visit http://getbootstrap.com/ to learn more about Bookstrap. For a detailed information about its grid system, click on http://getbootstrap.com/css/#grid.

As a brief explanation, the grid is composed of 12 columns; each named as "col-xs-1", "col-md-2", "col-lg-3", "col-xs-4", and so on and so forth. The column divisions are grouped in rows.

The grid system works for multiple devices. Please check the table below.

  Extra Small Devices(<481) Small Devices(≥768) Medium Devices(≥992) Large Devices(≥1200)
Container width Auto  
Class Prefix col-xs- col-sm- col-md- col-lg-

CSS

All css files are located in the /css.
 

- bootstrap/css
  • bootstrap.min.css - Generic bootstrap styles
- css
  • font-awesome.min.css - Font awesome file.
  • style.css - Default css file. You can change the theme from here
  • responsive.css - For custom responsive design
  • magnific-popup.css - For portfolio Image popup
  • animate.css - For css3 animation
  • bootstrap-dropdownhover.min.css - For Dropdown

Javascript

This template imports several JS files.

The Javascript  files are loaded in the footer. All files are loaded at the bottom to increase the site speed and rendering.

The Javascript files used in this template are the following:

  1. jquery.min.js - jQuery library
  2. bootstrap.min.js - Bootstrap JS  (bootstrap/js/bootstrap.min.js)
  3. jquery.magnific-popup.min.js - JS for Popup in Portfolio
  4. wow.min.js - JS for on load animation
  5. jquery.counterup.min.js - JS for counter
  6. waypoints.min.js - JS for counter
  7. bootstrap-dropdownhover.min.js - JS for dropdown
  8. main.js - Default JS for theme.

Changing Logo

Change a default logo by changing logo.png in the /images directory, or by opening the page and changing the following code.
